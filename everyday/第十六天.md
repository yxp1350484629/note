1.Python 日历模块calendar.monthrange 获取某一个月有多少天

```python
import calendar
monthRange = calendar.monthrange(2018, 10)
(0, 31)
```

`输出的是一个元组；`

`第一个元素，数字**0**是这个月的第一天是星期天（上一个月的最后一天为星期几(0-6)）``,星期天为``0；`

`第二个元素，数字**31**是这个月的天数；`
