# 希尔排序

```python
def hashsort(alist):
    a=len(alist)
    #得到第一次分组的gap值
    gap=a//2 
    #gap为1的时候就是排序结束的时候，而且alist的长度必须大于1
    while gap>=1 and a>1:
        for i in range(a):
            j=i
            #每组组员的间隔必须是组值，通过这个式子，可以得到所有的分组情况
            #这个核心就是依次分组，然后每个组之间按照插入排序进行操作。
            while j>=gap and alist[j-gap]>alist[j]:
                #如果大于就交换位置
                alist[j],alist[j-gap]=alist[j-gap],alist[j]
                #这个是要保证，后面的，可以跟前面的所有进行比较，依次互换
                j-=gap
        gap=gap//2
```
